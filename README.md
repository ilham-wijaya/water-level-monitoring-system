# IoT-Based Water Level Monitoring System

## Overview

This project is an IoT-based water level monitoring and prediction system developed as an undergraduate thesis.

The system integrates:

- ESP8266
- JSN-SR04T Ultrasonic Sensor
- Rain Sensor
- Python
- XGBoost Regressor

to monitor river water level in real time and predict future water levels.

---

## Features

- Real-time water level monitoring
- Rain detection
- Data logging
- Machine Learning prediction
- Early warning system
